分享一个免费制作文件外链的平台

发布于2021-09-14|更新于2023-09-26|博客教程
|字数总计:563|阅读时长:1分钟

唠唠闲话

之前介绍了 jsdelivr + github 搭建图床 的方法。其中 jsdelivr 是免费的 CDN 加速器，使用简单但文件大小受限。单个文件 ≤20m ，单个仓库 ≤50m。最近上传 pdf 文件超出了 jsdelivr 的限制，不得已转战其它平台。七牛云是一个不错的选择，每月 10G 免费流量，对小博客来说完全够用了。

下边就开始教程吧，本篇内容：

  • 搭建及使用
  • 告警设置
  • API 接口

参考文章

搭建及使用

点击进入官网，注册七牛云帐号，并用微信扫码实名。

创建空间

  1. 点击进入控制台，选择新建存储空间

  2. 填写空间名称，选择公开仓库，点击确定

  3. 系统自动生成了临时域名，有效期一个月，长期使用建议绑定自己的域名

制作文件外链

  1. 进入控制台，选择相应存储空间，点击文件管理和上传文件

  2. 不支持上传文件夹，但可以点击右侧的路径前缀实现文件夹效果

  3. 找文件时，输入前缀来模拟打开文件夹效果

  4. 找到文件，点击更多，选择复制外链

绑定域名

  1. 进入空间管理，点击仓库名

  2. 点击域名管理，选择绑定域名

  3. 填写域名信息，点击确定

注：七牛云用 http 访问，如果用 https 访问，流量不计入免费额度

  1. 点击刚绑定的域名，查看 CNAME 地址并复制

  2. 进入域名解析平台，添加 CNAME 记录，使用刚刚的地址

告警设置

下面介绍一下如何设置流量带宽告警配置。

  1. 进入 CDN 控制台，点击新建告警规则

  2. 填写规则，选择告警联系人，点击蓝色箭头添加

    注：初次使用，须设置告警联系人

  3. 点击新建联系人并填写信息

  4. 点击新建联系人组

  5. 选择联系人，点击蓝色箭头添加

可补充内容

  1. 用 picgo 或其他工具上传文件，更快捷方便。
  2. 七牛云的 API

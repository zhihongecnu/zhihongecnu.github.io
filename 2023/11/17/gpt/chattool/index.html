ChatTool

三月份 ChatGPT 开放 API 接口时写了这个工具，目前挂在实验室的 GitHub 账号 下。ChatTool 将 API 封装为简单易用的 Chat 对象，支持多轮对话，配置代理，异步处理数据，使用工具以及微调等功能。ChatTool 提供了一个简单易用的封装，本篇介绍其使用方式。

p.s. ChatTool 也写了个 Julia 版，不过没怎么维护

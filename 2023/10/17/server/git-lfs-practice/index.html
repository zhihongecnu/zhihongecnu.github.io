前言

LFS 全称 Large File Storage，是 Git 的一个扩展，用于存储大文件。其优点包括：

  1. 降低仓库大小，提高仓库 git clone 速度
  2. 仅在 checkout 到分支或 commit 时下载文件，历史提交的 chkpoint 或其他分支的 chkpoint 会主动忽略下载

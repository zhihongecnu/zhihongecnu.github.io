内容一览

安装主题

  1. 搭建博客框架，参看hexo | 博客框架搭建。

  2. 进入博客目录，下载 butterfly 主题，官方提供了两种方法
    方法一，用 git 将主题下载到 themes 目录

    方法二（同步不方便，不推荐），用 npm 将主题下载到 node_modules 目录

  3. 进入主题目录，复制配置文件 _config.yml 到博客主目录，并改名为 _config.butterfly.yml

  4. 修改 _config.yml ，将其中 theme 参数改为 butterfly

  5. 安装渲染工具

此时，用 hexo s 启动，就进入了 butterfly 主题。

注：命令 npm i 中，i 为 install 的缩写，但也有差异，参见npm i和npm install的区别。npm i 默认为局部安装，安装目录为 node_modules。

使用阿里 iconfont

参考文章
CSDN：Hexo博客之优雅使用阿里 iconfont 图标

一些图标 [FontAwesome] 上没有，比如 bilibili 小电视，这时可以用阿里的 iconfont 图标，使用教程如下

  1. 进入阿里 iconfont 寻找想要添加的图标，点击右上角购物车

  2. 依次点击新建项目 -> 添加项目 -> 确定

  3. 依次点击资源管理 -> 我的项目 -> 查看在线链接 -> 点击 css 文件

  4. 将文件保存到博客目录，例如放在 themes/butterfly/source/css/iconfont.css

  5. 引用 css 文件：在 head 标签中添加一行 <link rel="stylesheet" href="/css/iconfont.css">
    对 butterfly 主题，编辑 themes/layout/includes/head.pug，找到下边字段并修改（添加最后一行）

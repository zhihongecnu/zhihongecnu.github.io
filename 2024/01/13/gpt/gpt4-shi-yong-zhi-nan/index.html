GPT4 使用指南

发布于2024-01-13|更新于2024-01-25
|字数总计:546|阅读时长:1分钟

唠唠闲话

GPT3.5 网页端是免费，哪怕用 API 也是极其廉价。但为了更高的回答质量，通常会用到 GPT4. 这篇博客介绍在国内网络使用 GPT4 的两种方案。

方案一，基于 API 和 Lobe-Hub 项目

这个方案的门槛非常低，直接购买 API 就完事了。

  1. 访问网站：https://peiqi.shop/ 下单一个 API

  2. 购买后，会返回一个 sk- 开头的密钥，记下来

  3. 访问 Lobe-Hub 网站 https://chat.lobehub.com/

  4. 按下图流程，填入密钥，代理链接填：https://api.chatanywhere.tech/v1

  5. 直接访问代理链接 https://api.chatanywhere.tech，查询到 API 使用情况

Lobe-Hub 支持 GPT4 网页端的几乎所有功能，下图列举了其中一部分功能：

方案二，基于潘多拉项目

Lobe-Hub 的优点是开源，安全性，定制性强，也支持本地模型等。但 GPT4 API 的价格很高（约为 3.5 的100倍），单条对话通常1毛到5毛不等。如果多人使用，保守估计，一天40条对话就要10块钱，比买官方 GPT4 要贵一倍，更别说重度使用用户了。

方案二基于 pandora 项目，之间将 GPT4 的网页端转发到个人服务器，在国内网络通过个人服务器访问资源。

开通 GPT4 的方案，视频教程（油管）：不用国外虚拟信用卡，如何在iOS上开通ChatGPT Plus ｜附3分钟免费完成美区Apple ID申请

  1. 通过非港澳台的梯子，在 chat.openai.com 注册账号，需使用非国内邮箱（比如微软邮箱，谷歌邮箱）
  2. 注册后，登录网页端可以免费使用 GPT3.5 了
  3. 注册美区 ID 的苹果账号，在手机应用商店下载 ChatGPT（国区账号搜不到 ChatGPT）
  4. 在 ChatGPT 应用中登录刚才注册的账号，选择升级到 Plus 版本，通过支付宝进行支付。

在登录 ChatGPT 的网页端上，访问 chat.openai.com/api/auth/session ，获取到 token，记下来。

登录 Pandora 网站，这里给一个自建的 web： pandora123.cubenlp.cn 选择 accesstoken 方式登录，填入刚才的 token，即可使用 GPT4 网页端。
